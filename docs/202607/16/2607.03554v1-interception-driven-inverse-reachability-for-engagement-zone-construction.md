## 摘要
在对抗环境中，自主飞行器可能需要围绕发射位置未知的敌方追击者进行规划。本文提出了一种拦截驱动的逆向可达性框架，用于从观测到的单个追击者的拦截事件中直接推断其可行的发射区域。每次拦截都会对未知发射位置施加几何约束，对这些约束进行交集运算可得到有界集，在最大能力假设下该集合保证包含真实起点。通过追击者可达区域映射此推断集合，可产生具有明确最坏情况安全解释的确定性交战区域。一种概率扩展对追击者发射位置的不确定性进行建模，并生成用于风险感知规划的分级交战风险场。为加速定位，我们引入了一种针对牺牲代理的信息驱动规划器，该规划器选择轨迹以最大化可行发射区域的预期收缩。蒙特卡洛仿真表明，所提出的框架能快速减少发射位置的不确定性，并在仅少量牺牲部署后便可实现大幅缩短的安全轨迹。

## Abstract
In contested environments, autonomous vehicles may need to plan around adversarial pursuers whose launch locations are unknown. This paper presents an interception-driven inverse-reachability framework for inferring a feasible pursuer launch region directly from observed interception events for a single pursuer. Each interception induces a geometric constraint on the unknown launch location, and intersecting these constraints yields a bounded set guaranteed to contain the true origin under maximum-capability assumptions. Mapping this inferred set through the pursuer reachable region produces deterministic engagement zones with an explicit worst-case safety interpretation. A probabilistic extension models uncertainty in the pursuer launch location and yields graded engagement-risk fields for risk-aware planning. To accelerate localization, we introduce an information-driven planner for sacrificial agents that selects trajectories to maximize expected contraction of the feasible launch region. Monte Carlo simulations show that the proposed framework rapidly reduces launch-location uncertainty and enables substantially shorter safe trajectories after only a small number of sacrificial deployments.

### 2. 方法论

**核心思想**：追击者拦截一个代理表明其发射位置必然在其能力范围内可到达该拦截点，因此拦截点诱导一个圆形可达性约束（圆心在拦截点，半径 = 追击者最大航程 + 捕获半径）。多个拦截事件的约束取交集，得到有界的可行发射区域（Ppot）。再通过闵可夫斯基和（Minkowski sum）将该区域膨胀为追击者可达区域（RR），并逆向平移（考虑回避者运动）得到交战区域（EZ），在此区域外即为安全。

**关键技术细节**：

- **确定性EZ构建**（Section II.A）：
    - 给定可行发射区域 P，追击者可达区域 R = P ⊕ D(0, R+r)，其中 D 为圆盘，R 为最大航程，r 为捕获半径。
    - EZ：Z = R - νR ˆv_E，即把 RR 向回避者运动方向的反向平移 νR，ν 为速度比（evader/pursuer）。
    - 若 P 为矩形，则 R 为圆角矩形，EZ 亦然。
- **概率性EZ（PEZ）**（Section II.B）：
    - 发射位置服从分布 f(x_P)，则点 x 的可达概率 p_RR(x) = ∫_{||τ - x|| ≤ R+r} f(τ) dτ。
    - 回避者位置 x_E 的 PEZ 概率 p_EZ(x_E) = p_RR(x_E + νR ˆv_E)。
    - 当 f 为矩形均匀分布时，p_RR(x) 等于矩形与以 x 为中心、半径 R+r 的圆的交集面积比。
- **从拦截事件推断发射区域**（Section III）：
    - 无发射时刻信息：每个拦截点 x_int^(i) 形成圆盘 D(x_int^(i), R+r)，交集为 Ppot。
    - 有发射时刻信息：可计算追击者实际旅行距离 d_trav^(i) = v_P (t_int - t_launch)，约束圆盘半径为 d_trav^(i)+r，交集更紧。
    - 边界计算算法（Algorithm 1）：计算所有圆的交点，保留位于所有圆内的点，对每个圆上的顶点按角度排序，保留中点在所有圆内的弧段，得到由圆弧组成的凸多边形边界。
- **从 Ppot 生成 EZ/PEZ**：用符号距离函数（SDF）偏移实现闵可夫斯基和膨胀出 RR，再平移得到 Zpot；概率版本则用面积重叠计算。
- **安全轨迹规划**（Section IV.A）：B样条参数化轨迹，优化目标为最小化时间，约束包括动力学（恒定速度、曲率/转向率有界）以及避免进入EZ（硬约束）或限制PEZ概率低于阈值（风险感知）。
- **牺牲代理信息驱动规划**（Section IV.B）：
    - 无拦截时：最大化对初始猜测矩形区域的覆盖率 J0（轨迹覆盖的圆盘与矩形交集面积），以期首次拦截。
