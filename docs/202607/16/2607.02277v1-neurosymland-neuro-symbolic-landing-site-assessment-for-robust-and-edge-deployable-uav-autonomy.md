## 摘要
在非结构化环境中，安全着陆点评估仍是自主无人机部署的关键挑战，因为纯视觉学习方法在地形变化下性能常会下降，且在安全决策中透明度有限。我们提出了 NEUROSYMLAND，一个将轻量感知与显式安全推理相结合的神经符号着陆点评估系统。该框架从机载视觉输入构建概率语义场景图，并利用捕捉地形平坦度、障碍物间距与空间一致性的符号约束来评估候选着陆区域，从而在感知不确定性下实现结构化推理，同时保持边缘设备可执行性。在涵盖多种地形的 72 个模拟着陆场景中，NEUROSYMLAND 成功完成 61 次评估，优于四种竞争基线（37-57 次成功）。为评估可部署性，我们进一步开展了 100 次随机初始位姿的硬件在环试验，分析了端到端延迟、分阶段执行时间以及 CPU/GPU 利用率、内存占用和功耗等系统级指标。结果表明，在有界的边缘资源消耗下，鲁棒性与可解释性得到了提升。性能剖析显示，符号推理仅占端到端延迟的一小部分，主要计算成本来自感知与 PSSG 构建。这些结果证明了在资源受限的无人机边缘硬件上部署着陆点评估栈的可行性，所有源代码、数据集、提示词及符号规则优化示例均已开源发布。

## Abstract
Safe landing-site assessment in unstructured environments remains a key challenge for autonomous UAV deployment, as vision-only learning approaches often degrade under terrain variability and provide limited transparency in safety decisions. We present NEUROSYMLAND, a neuro-symbolic landing-site assessment system that integrates lightweight perception with explicit safety reasoning. The framework constructs a probabilistic semantic scene graph from onboard visual input and evaluates candidate landing regions using symbolic constraints capturing terrain flatness, obstacle clearance, and spatial consistency, enabling structured reasoning under perceptual uncertainty while maintaining edge-feasible execution. Across 72 simulated landing scenarios spanning diverse terrains, NEUROSYMLAND achieves 61 successful assessments, outperforming four competitive baselines (37-57 successes). To evaluate deployability, we further conduct 100 hardware-in-the-loop trials with randomized initial poses, profiling end-to-end latency, stage-wise execution time, and system-level metrics including CPU/GPU utilization, memory footprint, and power consumption. Results demonstrate improved robustness and interpretability with bounded edge-resource usage. Profiling shows that symbolic reasoning contributes only a small fraction of end-to-end latency, while the main computational cost arises from perception and PSSG construction. These results demonstrate the feasibility of deploying the landing-site assessment stack on edge-constrained UAV hardware, and all source code, datasets, prompts, and symbolic rule refinement examples are released in an open-source repository

### 2. 论文提出的方法论
- **核心思想**：分两阶段——“神经”侧构建显式世界模型，“符号”侧基于该模型进行确定性的安全推理与决策。运行时，神经网络不介入决策控制。
- **关键技术细节**：
    - **感知与世界模型构建**：使用 INT8 轻量化的 SegFormer-B0 进行语义分割，经几何后处理生成区域实例；各区域的语义、属性（平坦度、面积等）及区域间的空间关系（邻近、包含等）被组织成**概率语义场景图（PSSG）**，节点和边均携带概率置信度。
    - **符号规则推理**：采用 SCALLOP 概率逻辑程序。预定义的符号规则（如 `hazard(x) :- near_to(x,y), person(y)`）在 PSSG 的事实上执行推理，得出每个候选区域的安全/危险分数，并生成可追溯的证明路径。运行时仅执行固定的、经专家验证的规则集，不调用大语言模型。
    - **离线规则工程**：规则集由大语言模型（OpenAI o3）辅助起草，专家通过人在回路（HITL）方式审阅决策错误、修改规则，经迭代验证后固化。

- **算法流程**：单帧图像 → 语义分割与后处理 → PSSG 构建 → 符号规则推理得安全分 → 多帧一致性确认 → 任务加权排序 → 输出最优着陆点。

### 6. 论文的主要结论与发现
- **鲁棒性优势**：在 72 个场景中，NEUROSYMLAND 成功 61 次，超过最佳纯学习基线 PEACE（57 次）和 SafeUAV（47 次），最小障碍距离也更优。
- **边缘可行性**：端到端延迟约 1 秒（1043.3 ms），功率 12.3 W，内存 5.6 GB，Jetson Nano 上运行稳定。**符号推理仅占 1.9% 延迟（20 ms）**，主要成本来自感知与 PSSG 构建。
- **可解释性与可重配置**：决策能被追踪到具体的 PSSG 事实和规则（如 `near_to person`），通过仅修改符号规则而无需重训网络即可适应不同任务（安全、应急、救援）。
- **感知瓶颈**：失败案例主要源于语义分割错误，合理推理质量受限于世界模型准确度。

### 7. 优点
- **清晰的责任分离**：感知、世界建模、符号推理、任务排序完全解耦，极大增强了系统的透明度与可调试性。
- **运行时零大模型依赖**：离线借助大模型辅助编写规则，在线仅执行确定性的轻量逻辑程序，兼顾了智能性与边缘效率。
- **显式的世界模型接口**：以 PSSG 作为感知与推理的唯一桥梁，使得推理过程接地在结构化的、可审查的事实上，而非黑箱特征。
- **可重配置的决策逻辑**：更改任务模式只需调整排名权重或重写符号规则，无需收集数据重新训练，对实际运维意义重大。
- **详尽的边缘部署验证**：提供了分阶段时延、系统资源使用等硬数据，证明框架不是仅停留在概念层面。

### 8. 不足与局限
- **世界模型错误会确定性传播**：当前系统没有纠错机制，一旦感知将管道错分为可着陆区域，推理层无法补救，需提升感知可靠性或引入一致性校验。
- **场景动态性覆盖不足**：实验场景中缺乏移动障碍物，时序规则的优势未充分展现，限制了在有人物走动或环境快速变化时的适用性。
- **仅评估站点而非完整着陆**：实验聚焦于着陆点选择正确性，未集成轨迹规划、控制等模块做闭环飞行，距真实任务仍有距离。
- **规则工程仍依赖专家**：虽有大模型辅助，但最终仍需人审阅和迭代，当场景类别激增时，维护成本可能上升。
- **PSSG 构建延迟较高**：平均 557 ms，几乎占端到端一半，对于需要更快反应速度的场景（如紧急避障）略显迟缓。
